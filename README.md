# Pranavnath S – Professional Resume Website

This is a clean and professional personal resume website built using HTML and CSS.
It highlights academic background, legal experience, research publications, and
corporate law interests.

## 🔹 Features
- Minimalist white & grey professional design
- Downloadable resume PDF
- Optimized for recruiters and law firms
- Mobile responsive layout

## 🔹 How to Use
1. Download or clone this repository
2. Place your resume PDF inside the `assets` folder
3. Open `index.html` in a browser

## 🔹 Deployment
You can deploy this site for free using:
- GitHub Pages
- Netlify
- Vercel
